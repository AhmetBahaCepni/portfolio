Projects

cub3D

3D RayCaster Game

cub3D is a 3D game written in C. It uses the raycasting algortihm which is used in the first FPS game Wolfenstein 3D and uses minilibx library to draw to the screen. The goal is to make a dynamic view inside a maze.

If we need to explain raycasting breifly, after creating the 2d world map, we cast rays from the player's position to the walls. Then we calculate the distance between the player and the wall for each ray. this way we can draw closer walls bigger and further walls smaller. This way we create a 3D effect.

Minishell

Own Unix Shell

Minishell is a simple shell written in C. The goal is to make a shell like bash or zsh. Minishell can handle commands, pipes, redirections and heredocs just like a real shell.

We used processes and many system calls in this project such as fork, execve, dups, wait, signals, etc. Processes needed to execute non-built-in commands without terminating the shell.
